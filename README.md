# Burrow

## Installation

Burrow can be installed on both **macOS** and **Linux**. Before installing, ensure that the following prerequisites are available in your environment:

- **Go** 1.25 or later
- **Make** 3.81 or later

Once the prerequisites are satisfied, run the following command to install Burrow:

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/TypingHare/burrow/HEAD/script/install.sh)"
```
